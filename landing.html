How to Make a Github.io Portfolio

Using a precongifured responsive HTML/JS template I lead students through editing and setting up a portfolio website by editing HTML and CSS.

Why did we use a preconfigured template?

Webpages that work well on mobile, desktop, tablets and screens on many different sizes need responsive layouts. In today's day and age HR recruiters will be browsing on smart phones, their office computer, etc. This template came from Pixelarity.com and is fully licensed for use by the Instructor Mr. Siefen (me!)

What are the key parts to this project?

  • HTML or Hypertext Markup Langauge
    • Contains element tags for common items like paragraphs, lists, headings, images, etc.
    • Can be edited in any text editing program! Viewable in any browser
  • CSS of Cascading Style Sheets
    • Controls the look of the webpage with Style properties
    • Sometimes is contained in a seperate file, but can be controlled in-line in html.
  • JS and JQuery
    • Handles the responsive parts of the website
  • Github and Github.io
    • Github - Webhosting of version controlled code, documents, files.
    • Github.io now called github pages, is a website hosting tool usable with a public github repository containing your website files.

What skills would you learn?

  • HTML Content Editing
  • CSS styling basics
  • Linking multiple webpages into a website
  • Creating a 404 page
  • Using version control
  • Hosting a website online
  • Creating an online portfolio

How is this applicable to the real world?

In my time as a "Webmaster"(look it up if you've never heard the term) I learned that all you need to do is see into the future, predict what the client really wants instead of that they tell you and make it happen. Obviously this is never the case so being able to parse HTML files that are "unknown", changing content and updating new pages from a template is key. Many webpages do not need any backend services or databases to store data. These websites simply have content to display and maybe use some CSS and JS to make it look pretty and work nicely. But the content is all HTML. This project teaches you to edit HTML to your whims. Back in the 90s you could've made 50k a year just doing that.
