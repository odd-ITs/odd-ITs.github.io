Creating an Incremental Game in Scratch

In class project, using Scratch, some sound effects, basic sprites. The purpose is to teach students about event handling and basic game mechanics.

Creating a Classroom Sign in Easel

Using the X-Carve CNC Routers, Easel CAM software and some good old fashioned elbow grease, we created new classroom signs for teacher's all over the district. Students designed signs, routed them on the CNC and then filled the signs with resin, sanded, and then stained them to perfection.

FRC Robotics Team 453

This is the LCPS Combined High School robotics team. We have students from both LCN and LCHS. Our team builds large scale mobile robots to compete in new challenges every year.

Wanda Series Robots

In partnership with RGGI and Yellow Fox llc, we created a UVC sanitizing robot called Wanda. Wanda SD is a remote controllable robot with an Android and iOS app. Wanda SA is a fully autonomous version currently in the works.

Hi-5inder

Created for the Grizzhacks 3 Hackathon, we created a social media app focused around giving hi-5s. This used gps data, firebase, and other technologies we learned on the fly in 24 hours.
